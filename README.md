# Weather App

A modern, responsive weather application built with React that provides real-time weather information for any city worldwide.

## Features

- Real-time weather data from OpenWeatherMap API
- Search weather by city name
- Get weather for current location using geolocation
- Multi-language support (English & Thai)
- Detailed weather information (temperature, humidity, wind speed, pressure, visibility)
- Sunrise and sunset times
- Loading skeleton for better UX
- Recent searches saved locally
- Modern Sky & Ocean theme design
- Responsive layout

## Installation

```bash
npm install
```

## Usage

```bash
npm start
```

Open [http://localhost:3000](http://localhost:3000) to view it in your browser.

## Build

```bash
npm run build
```

Builds the app for production to the `build` folder.

## Technologies

- React 18
- OpenWeatherMap API
- CSS3 with modern gradients
- Local Storage for persistence
